UUID Panel Node
http://drupal.org/sandbox/Hydra/1621954
====================================


DESCRIPTION
-----------
This module replaces the add node by nid function from Panels with an add node by uuid functunallity. You'll be able to select the node as usual but instead of catching the nid, this module will catch the uuid of the node and safe it in your pane.



REQUIREMENTS
------------
Drupal 7.x
Panels - http://drupal.org/project/panels
UUID - http://drupal.org/project/uuid


INSTALLING
----------
1. To install the module copy the 'uuid_panel_node' folder to your sites/all/modules directory.

2. Read more about installing modules at http://drupal.org/node/70151


CONFIGURING AND USING
---------------------
1. Their is nothing else to configure! Go into your panels and add a content to a pane. UUID Node should be available.
